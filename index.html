JavaScript

Comparison Operators

==

(Equal)

compares two values, independent of type and returns true if VALUES are the same

===

(Strict Equal)

compares two values and returns true if both value and type are the same

!=

(Not Equal)

checks if two values are not equal independent of type

!==

(Strict Not Equal)

checks if two values AND types are not equal

&&

(Logical AND)

evaluates to true if both values are true, otherwise false

||

(Logical OR)

evaluates to true if either value is true, otherwise false

Conditional Statements

if statements

if something is true, do this. otherwise, dont do it.

if... else statements

similar to if, but gives another option if condition isn't met.

if... else if... else statements

handles multiple conditions and defaults to else if other conditions arent met.
